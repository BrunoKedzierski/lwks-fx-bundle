All of the effects in this zip file will run on all versions of Lightworks, with the exception of PolyMask14 and PolyMask16.
The versions of those effects in this folder will run on any version of Lighworks except for Windows 32 bit or 64 bit version 14.0 and earlier.  Versions that will run on older Windows Lightworks have been placed in the "Legacy Windows" folder.
